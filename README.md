# Bakery Management System (PHP & MySQL)

## Overview
This project is a basic bakery management and online ordering system developed using PHP and MySQL.  
It simulates a real-world bakery website with both user and admin functionalities.

## Features
- User registration and login system
- Admin panel
- Product management
- Shopping cart system
- Order tracking
- Password reset functionality
- Database integration

## Technologies Used
- PHP
- MySQL
- HTML5
- CSS
- XAMPP (Local Development Environment)

## Database
The database schema is included in the `veritabani.sql` file.

## Purpose
The purpose of this project is to understand:
- Backend development fundamentals
- Database connections
- CRUD operations
- Authentication systems
